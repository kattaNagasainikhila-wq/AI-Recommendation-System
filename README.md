# 🎯 AI Recommendation System using TF-IDF and Cosine Similarity
## 📌 Project Overview
This project is a simple AI-powered Recommendation System developed as part of my Artificial Intelligence Internship at DecodeLabs.
The system takes user interests as input and recommends the most relevant courses based on similarity matching using TF-IDF Vectorization and Cosine Similarity.
It demonstrates the fundamental concepts behind recommendation engines used by platforms such as Netflix, Amazon, YouTube, and Spotify.
## 🚀 Features

✅ User Interest-Based Recommendations

✅ TF-IDF Vectorization

✅ Cosine Similarity Matching

✅ Top 5 Personalized Recommendations

✅ Match Percentage Calculation

✅ Course Recommendation Ranking

✅ Simple and Interactive User Interface

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-Learn
* TF-IDF Vectorizer
* Cosine Similarity

## ⚙️ How It Works

### Step 1

The user enters their interests.

Example:

python ai machine learning

### Step 2

The system converts user interests and course skills into numerical vectors using TF-IDF.

### Step 3

Cosine Similarity calculates how closely each course matches the user's interests.

### Step 4

Courses are ranked according to similarity scores.

### Step 5

The top recommendations are displayed to the user.
